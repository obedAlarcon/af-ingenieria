# AF Ingeniería

Sitio web corporativo desarrollado para AF Ingeniería, empresa especializada en soluciones de seguridad electrónica, automatización, redes y soporte tecnológico.

---

## Descripción

Este proyecto corresponde al desarrollo del sitio web institucional de AF Ingeniería. Su objetivo es presentar los servicios de la empresa, facilitar el contacto con los clientes y ofrecer una experiencia de navegación moderna, rápida y adaptable a cualquier dispositivo.

## Características

- Diseño responsivo.
- Interfaz moderna y optimizada.
- Formulario de contacto con PHPMailer.
- Integración con WhatsApp.
- Navegación fluida de una sola página.
- Optimización de imágenes y recursos.
- Código organizado y modular.

## Tecnologías

| Tecnología | Uso |
|------------|-----|
| HTML5 | Estructura |
| Tailwind CSS | Estilos |
| JavaScript | Interactividad |
| PHP | Procesamiento del formulario |
| PHPMailer | Envío de correos |
| XAMPP | Desarrollo local |

## Estructura del proyecto

```text
af-ingenieria/
├── assets/
├── pages/
├── PHPMailer/
├── config.php
├── enviar.php
├── index.html
└── README.md
```

## Instalación

1. Clonar el repositorio.
2. Copiar el proyecto en `xampp/htdocs`.
3. Crear el archivo `config.php`.
4. Iniciar Apache.
5. Acceder desde:

```text
http://localhost/af-ingenieria
```

## Configuración

El archivo `config.php` contiene las credenciales SMTP y no forma parte del repositorio.
